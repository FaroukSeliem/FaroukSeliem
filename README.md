### 🌍 Global Health Data Analytics | Power BI

> Three decades of global mortality data — synthesizing **1.46 billion death records** across 204 countries to uncover the primary drivers of death worldwide.

| KPI | Value |
|-----|-------|
| 🌐 Countries Covered | 204 |
| 📋 Death Records | 1.46 Billion |
| 📅 Time Span | 1990 – 2019 |
| 📈 Deaths Growth | 44M → 54M annually |

**🔍 Key Highlights:**
- Identified the **"Big Three"** killers: Cardiovascular diseases, Cancers & Chronic Respiratory diseases
- Revealed heavy mortality concentration in **China, India, and the USA**
- Tracked a steady annual death rise from **~44M (1990)** to **54M+ (2019)**
- Built a high-contrast dashboard with KPI cards and dual-axis charts for scannability
